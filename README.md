# <div align="center">**Musik Shop — магазин музыкального оборудования**</div>

## О проекте
<span style="font-size: 16px;">**Musik Shop**</span> — это веб-приложение, представляющее собой интернет-магазин для продажи музыкального оборудования. В перечень товаров для продажи входят: музыкальные инструменты, студийное и звуковое оборудование и другие товары музыкального профиля. Наша цель — упрощение покупки различных товаров, связанных с музыкальной сферой, как для любителей, так и для профессионалов. Musik Shop работает с ведущими компаниями в этой области, такими как: Fenter®, Cord®, Bibson®, Goland®, Casico® и так далее.
<span style="font-size: 14px;">**Наша целевая аудитория**</span> — новички и любители, которые только открывают для себя мир музыки и хотят приобрести достойные товары по приемлемой цене. Но и профессионалы найдут для себя нужное оборудование. Также наш магазин сотрудничает с различными студиями по поставке новейшего и самого продвинутого студийного оборудования.
<span style="font-size: 14px;">**Главными ососбенностями**</span> нашего веб-приложения являются возможность продвинутой фильтрации товаров для их поиска, а также возможность добавления товаров в корзину для продолжения просмотра каталога магазина и последующего перехода в нее, чтобы завершить оформление заказ. Все эти функции предназначены для удобного взаимодействия пользователя с сайтом.

## Технологический стек
### Frontend:
- **Vue.js** — JavaScript фреймворк, его преимущество в низком пороге вхождения, гибкости а также в производительности.
- **Vue Router** — глубокая интеграция с Vue.js, продвинутое сопровождение путей и поддержка режимов HTML5 History/hash.
- **PINIA** — state-менеджмент библиотека для хранения глобальных данных
### Backend:
- **Node.js** — среда выполнения JavaScript, выбранная за высокую производительность и за единый язык с Frontend-ом
- **Express.js** — это минималистичный, быстрый и гибкий веб-фреймворк для Node.js, являющийся инструментом для создания backend-части веб-приложений и API на JavaScript.
- **MySQL** — бесплатная система управления базами данных, простая в использовании и с высокой производительностью.

## <div align="center">Схема данных</div>

### User (Пользователь)

| Поле          | Описание                              |
| ------------- | ------------------------------------- |
| id            | Уникальный идентификатор пользователя |
| email         | Адрес почты, используется для входа   |
| username      | Отображаемое имя пользователя         |
| password_hash | Захешированный пароль                 |
| created_at    | Дата и время регистрации              |

### Product (Товар)

| Поле        | Описание                                     |
| ----------- | -------------------------------------------- |
| id          | Уникальный идентификатор товара              |
| name        | Название товара (3-100 символов)             |
| description | Краткое описание товара                      |
| price       | Цена товара                                  |
| image_url   | Ссылка на изображение товара                 |
| category    | Категория товара                             |
| stock       | Количество на складе                         |

## Ключевые эндпоинты API

### Аутентификация

| Метод | Эндпоинт                | Описание                         |
| ----- | ----------------------- | -------------------------------- |
| POST  | `/api/auth/register`    | Регистрация нового пользователя  |
| POST  | `/api/auth/login`       | Вход в аккаунт по email и паролю |
| POST  | `/api/auth/logout`      | Выход из аккаунта                |
| GET   | `/api/auth/check-email` | Проверка уникальности email      |

### Товары

| Метод  | Эндпоинт              | Описание                                     |
| ------ | --------------------- | -------------------------------------------- |
| GET    | `/api/products`       | Получить список товаров с фильтрацией        |
| GET    | `/api/products/:id`   | Получить детальную информацию о товаре по ID |
| POST   | `/api/products`       | Создать новый товар                          |
| PATCH  | `/api/products/:id`   | Обновить данные товара                       |
| DELETE | `/api/products/:id`   | Удалить товар                                |
